## 常见字形变体

部分小写希腊字母在公式中有两种常用字形：

| 字母 | LaTeX 代码 |
| --- | --- |
| Epsilon 变体 | `\varepsilon` |
| Theta 变体 | `\vartheta` |
| Pi 变体 | `\varpi` |
| Rho 变体 | `\varrho` |
| Sigma 词尾形式 | `\varsigma` |
| Phi 变体 | `\varphi` |

> 标准 LaTeX 没有为所有外形与拉丁字母相同的大写希腊字母提供单独命令。例如 Alpha、Beta 和 Epsilon 的大写形式通常直接写作 `A`、`B` 和 `E`。
