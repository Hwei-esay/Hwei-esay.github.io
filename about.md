这是一个面向长期积累的个人知识站点，适合放四类内容：

1. 100 到 200 字左右的短博客，记录阶段思考和近期项目。
2. 十几个字的感悟短句，快速留下灵感。
3. 可复用的教程与踩坑记录，避免重复造轮子。
4. 学术报告与讲座笔记，方便回看和二次整理。

我希望这里不是“堆内容”的地方，而是一个能持续提炼方法、观点和经验的小型知识库。

如果你也在做类似的积累，欢迎通过 GitHub 或邮件交流。
